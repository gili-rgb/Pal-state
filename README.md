# pal-state — שכבת מצב חי ל-Pal Group

הריפו הזה הוא **מקור האמת למה שמשתנה**: סטטוס פרויקטים, גרסאות סקילים, מה פתוח, ומה הוחלט.
הוא לא מחליף את הזיכרון המובנה ולא משכפל אותו.

## חלוקת אחריות (קריטי — אל תשבור)
- **זיכרון מובנה** = מה שלא משתנה: כללי ברזל (NAP, קישורים אסורים, טרמינולוגיה, פרוטוקול SKILL, מספרי Yoast), מי גיל, פרוטוקולים. נטען אוטומטית.
- **ריפו זה** = מה שמשתנה: מצב, גרסאות, פתוחים, החלטות. נטען בקריאה אקטיבית בתחילת שיחה.
- **קבצי SKILL** = איך מבצעים כל דבר. ב-`/mnt/skills/user/`.

עדכון כלל ברזל הולך לזיכרון המובנה בלבד. הריפו רק רושם ב-DECISIONS שהכלל עודכן, בלי לשכפל תוכן.

## סדר טעינה בתחילת כל משימה לא-טריוויאלית
1. `git clone --depth 1 https://github.com/gili-rgb/Pal-state.git /home/claude/pal-state`
2. קרא `STATE.md` — תמונת מצב מלאה
3. קרא `OPEN.md` — מה פתוח כרגע
4. `DECISIONS.md` נקרא רק כשצריך להבין *למה* הוחלט משהו
